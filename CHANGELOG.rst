Changelog
==========

**0.3.0** (02.03.2026)
------------------------
- Data pipelines for GLiM & GLHYMPS now pull from online sources, eliminating the need for local data storage.
- Data pipelines for rock properties and hydraulic gradient (Germany) datasets still rely on local data.

**0.2.0** (02.02.2026)
------------------------
- Added data pipelines for GLiM, GLHYMPS, rock properties, and hydraulic gradient (Germany) datasets.
- Backend pipeline: Looking up local parameters
- Backend pipeline: Analysis run returns derived quantities and ATES KPIs

**0.1.0**
------------------------
- Complete refactoring of previous GeoNinja
- Dynamic definition of parameters and analysis results, validated through a common data contract.
- Two visual layers: OpenStreetMaps and Satellite imagery.
